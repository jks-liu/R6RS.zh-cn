## PDF（目前处于未完成状态）
### 生成PDF
在[latex/document](/latex/document/)目录执行：

``` bash
make -i
```

参数`-i`表示忽略错误，遇到错误请直接回车。**请帮助我**：我不清楚为什么会有这些错误，请知道的人告诉我。我的编译环境是Raspberry Pi，使用的是TexLive。
